
# 🔵 TP — Clustering K-means & FCM

## 📌 Description
Travaux pratiques sur les algorithmes de clustering K-means et 
Fuzzy C-Means (FCM) réalisés à ENICarthage.

## 🎯 Objectifs
- Comprendre le principe du clustering non supervisé
- Implémenter l'algorithme K-means
- Implémenter le Fuzzy C-Means (FCM)
- Comparer les résultats des deux algorithmes
- Visualiser les clusters obtenus

## 📁 Contenu
- Notebooks Jupyter avec les implémentations complètes
- Datasets utilisés pour l'analyse

## 🛠️ Technologies utilisées
- Python
- Pandas
- Scikit-learn (KMeans)
- Scikit-fuzzy (FCM)
- Matplotlib
- Seaborn
- NumPy
